Signature Beschreibung

Signature ist eine Markup-Language, die einen Relativ (starken) einfachen Aufbau hat. Der Syntax wird in HTML umcompiled damit jeder es öffnen kann. Man muss für jede Zeile eine Definition machen. z.B.:

1<<<;
d:Hallo Welt;
1>>>;

Es ist wichtig, dass keine Abstände nach ";" kommen

Schlüsselbegriffe:

home; -> Im Notebook setzt er dass aufs Rootverzeichnis; Erzeugt einen Link auf die "Home" Seite
style: Arial; -> Hier kann man seine Font-Family einstellen
name: name; -> Name der Datei/der Notiz
1<<<; -> Macht eine Reihe
<<<; -> Macht eine Reihe zu und dazu noch eine auf, sie macht aber letztere nicht wieder zu
1>>>; -> Macht eine Reihe zu
spac; -> Ein HTML break
&scs -> Semicolon &ncs -> Colon
d: Text;
db: dicker Text;
dh: Überschrift;
dhb: dicke Überschrift;
g: grüner Text;
gb: grüner dicker Text;
gh: grüne Überschrift;
ghb: grüne dicke Überschrift;
r: roter Text;
rb: roter dicker Text;
rh: rote Überschrift;
rhb: rote dicke Überschrift;
b: blauer Text;
bb: blauer dicker Text;
bh: blaue Überschrift;
bhb: blaue dicke Überschrift;
code: code; -> Schriftart für Code oder Befehle forw: html-code; -> Da dieser Syntax in HTML-Code übersetzt wird kann der eingetragene Code geforwardet werden

Es gibt bis jetzt noch keine IDE (oder - Extension) für diese Sprache, weitere Informationen können in GitHub gefunden werden.
