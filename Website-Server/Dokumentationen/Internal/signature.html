Signature Beschreibung

Signature ist eine Markup-Language, die einen Relativ einfachen Aufbau hat. Der Syntax wird in HTML umcompiled damit jeder es öffnen kann. Man muss für jede Zeile eine Definition machen. z.B.:

1<<<;
d:Hallo Welt;
1>>>;

Es ist wichtig, dass keine Abstände nach ";" kommen

Schlüsselbegriffe:

home; -> Im Notebook setzt er dass aufs Rootverzeichnis; Erzeugt einen Link auf die "Home" Seite
style: Arial; -> Hier kann man seine Font-Family einstellen
1<<<; -> Macht eine Reihe
<<<; -> Macht eine Reihe zu und dazu noch eine auf, sie macht aber letztere nicht wieder zu
1>>>; -> Macht eine Reihe zu
d: Text; -> Normaler Text
db: dicker Text -> Dicker Text

Es gibt bis jetzt noch keine IDE (oder - Extension) für diese Sprache, weitere Informationen können in GitHub gefunden werden.
