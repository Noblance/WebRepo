SSH Einrichtung

  1. SSH mit #apt install openssh-server herunterladen
  2. Ein Directory bei den SSH-Host (sollte empfhelend in sudoers drin sein) errichten mit dem Namen ".ssh"
  3. Auf der Client-Maschine ein RSA-Key-Pair erstellen mit #ssh-keygen
  4. Nun den Schlüssel von der Client Maschine zur Hostmaschine übertragen mit #scp
    • Windows: #scp $env:USERPROFILE/.ssh/id_rsa.pub <ssh-client>@<ip-adresse>:~/.ssh/authorized_keys
  5. Nun SSH "versiegeln", indem man den ROOT-Acces verbietet, den Port auf einen anderen legt und nur RSA_Keys erlaubt
    • #nano /etc/ssh/sshd_config
    • Port <port>
    • PermitRootLogin no
    • PasswordAuthenfication no
  6. Der Server kann nun mit #ssh <ssh-client>@<ip-adresse> -p <port> erreicht werden
  7. Hinweis: Wenn mehrere Schlüssel-Paare vorhanden sind kann das mit -i <Directory> spezifiziert werden
Bei allen Befehlen kann/muss sudo verwendet werden.
