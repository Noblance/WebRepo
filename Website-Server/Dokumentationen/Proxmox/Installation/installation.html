Proxmox Installation

  1. Die Proxmox VE ISO-Datei von Proxmox holen.
  2. Die Datei mit einen Tool das ISO-Datein auf USB-Stick oder DVD flashed (z.B.: Etcher oder Rufus; Empfehlung Rufus)
    • Es kann sein das der USB-Stick (oder DVD) als mehrere Laufwerke vorkommt
  3. In das BIOS des PCs/Servers/Laptops gehen (F1, F2, Esc o.Ä.) und die Bootorder (unter der Registerkarte Boot) als erstes auf USB legen. Virtualisierung sollte auch noch eingeschaltet werden (Advanced)
  4. Nun den Installations Prozess durchlaufen
    • Beim Hostnamen nach dem Hostnamen ".local" anhängen damit er zu keiner domain hinzugenommen wird, sofern es ein Homelab ist
    • Eine fixe IP-Adresse zuweisen die im Heimnetzwerk nicht unter DHCP vergeben wird
  5. Wenn das geschaft ist kann der "Server" unter der Web-UI mit den Port 8006 erreicht werden
